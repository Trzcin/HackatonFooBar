# 15 Minute City

Aplikacja internetowa pomagającą określenie atrakcyjności miejsca zamieszkania pod względem założeń miasta 15 minutowego.

## Instrukcja uruchomienia

### Frontend

```
cd frontend
npm install
npm run dev
```

### Backend

```
cd backend
docker-compose up
```
